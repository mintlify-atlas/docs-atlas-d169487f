## Why uniku?

  <Card title="Universal Runtime Support" icon="globe">

  <Card title="Six ID Formats" icon="fingerprint">

  </Card>

  </Card>
  <Card title="Modern Standards" icon="star">

## Quick Example

```typescript
import { uuidv7 } from 'uniku/uuid/v7'

const id = uuidv7()
// => "018e5e5c-7c8a-7000-8000-000000000000"

// Time-ordered: IDs sort by creation time
const [first, second, third] = [uuidv7(), uuidv7(), uuidv7()]
console.log(first < second && second < third) // true
```
